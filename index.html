Zonaprop Price per m² Chrome Extension

Download the following files to install the extension:

1. manifest.json
2. content.js
3. icon.svg

Installation Instructions

  1. Create a new folder named "zonaprop-price-per-m2" on your computer
  2. Save all downloaded files into this folder
  3. Open Chrome and go to chrome://extensions/
  4. Enable "Developer mode" (toggle in top-right)
  5. Click "Load unpacked"
  6. Select the folder you created
  7. Visit zonaprop.com.ar to see it in action!

The extension will automatically calculate and display the price per square meter in green text next to each property's price.
